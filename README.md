# AI CIV

## 游戏概述
这是一个人工智能模拟人类文明发展的游戏，让两个AI模型控制的文明从原始时期发展到未来。

游戏内容在master分支。

## 功能特点
- 两个AI文明相互竞争发展
- 支持阿里云百炼模型，可以配置不同模型看海
- 模拟文明的多个维度：资源、人口、军队、科技、忠诚度、文化
- 不同纪元会有随机的不同时代事件，需要AI付出资源来得到奖励或规避惩罚
- 每个回合AI都会有一些AP（行动点）来做决策
- 支持AI之间的外交，现在包括贸易和战争
- 自动保存游戏状态
- 详细的游戏日志和最终总结
