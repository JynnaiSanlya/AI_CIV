# AI Civilization Simulation - 版本 0.4.0

## 发布日期
2026年3月18日

## 新功能

### 1. 多文明支持
- **支持3+ AI文明**：游戏现在支持最多8个文明，通过`civ_configs`参数配置
- **动态文明管理**：文明存储在字典中，而非硬编码变量
- **自动UI适配**：Web界面根据活跃文明动态生成文明面板
- **全文明交互**：文明可以与所有其他文明交互（贸易、战争、文化影响）

### 2. 性能优化
- **并行回合执行**：文明回合现在使用线程并行处理
- **减少等待时间**：不再顺序执行文明回合
- **提高回合处理速度**：多文明游戏进程更快

### 3. 增强AI决策
- **多文明感知**：AI现在考虑所有其他文明的信息
- **全面世界观**：AI接收所有已知文明的详细信息
- **改进战略规划**：AI可以平衡与多个对手的关系和策略
- **增强提示设计**：包含所有相关文明数据的更好提示

### 4. Web界面改进
- **动态UI生成**：根据活跃文明动态创建文明面板
- **响应式设计**：UI适应不同数量的文明
- **增强数据可视化**：更好地展示多文明交互
- **改进最终结果**：更好地处理多文明评分和获胜者判定

## 技术改进

### 1. 代码架构
- **重构游戏核心**：改进代码组织，提高可维护性
- **字典基础存储**：用字典替换硬编码的文明变量
- **更好的可扩展性**：代码现在更适合未来功能扩展
- **改进错误处理**：更好地处理多文明边缘情况

### 2. AI系统
- **增强AI API**：更新AIController以处理多个对手
- **更好的提示工程**：更全面的AI决策提示
- **改进响应解析**：更好地处理AI响应
