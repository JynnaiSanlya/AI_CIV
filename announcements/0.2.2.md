# 版本 0.2.2 更新公告

发布日期：2026-03-15

## 主要更新

### AI决策改进
- 增强了资源收集提示，鼓励AI在没有其他行动时收集资源
- 鼓励AI尽可能最大化收集资源，提高资源利用率
- 强调资源收集无成本，降低AI的决策犹豫

### 外交系统重构
- 将随机触发的贸易和战争改为由AI驱动决策
- 添加了专门的外交决策方法 `get_diplomacy_decision`
- AI现在基于游戏状态决定是否提议贸易、宣战或维持和平
- AI指定具体的贸易提议和请求细节，包括资源、人口和科技的交换量
- 每回合只允许一次贸易和一次战争（基于文明优先级）
- 完善了贸易和战争的规则说明，帮助AI做出更明智的决策

### 增强AI战争兴趣
- 改进了战争收益描述，清晰展示潜在回报
  - 资源掠夺：最高可获得对手40%的资源
  - 人口掠夺：最高可获得对手40%的人口
  - 文化掠夺：最高可获得对手20%的文化
  - 忠诚度提升：获胜者获得忠诚度奖励
- 平衡了战争收益与风险，避免AI盲目宣战
- 添加了战略考虑因素：
  - 军事优势：更高的军事实力增加获胜机会
  - 时代优势：更高时代拥有更强的军事乘数

### 最终分数结算改进
- 在最终总结中添加了文化分数的详细显示
- 确保分数计算与显示的一致性
- 提高了分数透明度，让玩家清楚了解各属性对最终分数的贡献
- 文化权重为10%，与资源权重相同

### Web UI最终结果显示
- 添加了专门的最终结果显示区域
- 显示两个文明的最终分数对比，使用大字体突出显示
- 清晰的获胜者公告，包括获胜文明名称
- 获胜优势显示，展示获胜方的领先分数
- 美观的CSS样式设计，提升视觉效果
- 支持平局情况处理，显示平局公告

## 技术更新
- 优化了AI提示结构，提高了AI决策质量
- 改进了外交决策的提示工程
- 增强了游戏结束时的状态处理
- 完善了Web UI的游戏结束逻辑
- 优化了最终结果的数据传递

## 游戏体验
- 游戏变得更加策略化，AI做出更明智的决策
- 外交互动更加丰富，AI根据自身情况主动提出贸易或宣战
- 战争系统更加平衡，AI会权衡战争的收益和风险
- 游戏结束时提供清晰、美观的结果总结
- 提高了游戏的透明度，让玩家更好地理解游戏机制

## 如何升级

直接运行最新版本的游戏即可，无需额外配置。

```bash
python web_gui.py
```

访问 http://localhost:8000 查看Web图形界面。
