# 版本更新公告 - 0.1.4 (2026-03-14)

## 更新内容

### 1. 战争收益增强
- 掠夺上限从20%提高到40%，使战争更具吸引力
- 掠夺系数计算公式优化，提高了实际掠夺量
- 确保战争不会将对方资源或人口掠夺至负数

### 2. 忠诚度惩罚增强
- 资源不足时的忠诚度惩罚显著提高
- 忠诚度损失计算公式从 `min(10, unfed_people // 50 + 1)` 调整为 `min(20, unfed_people // 25 + 2)`
- 资源短缺对文明稳定的影响更加明显

### 3. AI行动处理优化
- 当文明行动点为0时，自动停止执行后续行动
- 减少了不必要的AI决策尝试
- 提高了游戏运行效率
- 行动点耗尽时显示明确提示

## 技术更新

### 1. 代码优化
- 修改了 `attack` 方法，增强了战争收益
- 更新了 `update_turn` 方法，增强了忠诚度惩罚
- 优化了 `handle_civilization_turn` 方法，添加了行动点检查

### 2. 游戏机制调整
- 战争系统更加平衡，鼓励战略决策
- 资源管理更加重要，影响文明忠诚度
- AI行为更加合理，避免无效决策

## 游戏机制

### 战争系统
- 获胜方可以掠夺失败方最多40%的资源和人口
- 掠夺量基于剩余军事力量
- 确保失败方不会被掠夺至负数
- 战争会对双方忠诚度产生影响

### 忠诚度系统
- 资源不足时忠诚度会显著下降
- 人口饥饿会导致更大的忠诚度惩罚
- 忠诚度影响文明的稳定性和发展

### AI决策优化
- AI会在行动点耗尽时自动停止行动
- 减少了无效的决策尝试
- 提高了游戏运行效率
