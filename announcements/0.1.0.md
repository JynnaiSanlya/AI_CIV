# 版本更新公告 - 0.1.0 (2026-03-14)

## 初始版本发布

### 游戏概述
AI Civilization Simulation是一个人工智能模拟人类文明发展的游戏，让两个AI模型控制的文明从原始时期发展到未来。

### 核心功能

#### 1. 文明系统
- 两个AI控制的文明（Atlantis和Eldorado）
- 文明属性：资源、人口、军队、科技、忠诚度
- 自动时代升级机制

#### 2. AI模型支持
- 支持阿里云百炼qwen-flash和qwen-plus模型
- 可配置使用不同AI模型

#### 3. 游戏机制
- 回合制游戏循环
- 行动系统：发展科技、建造军队、增长人口、收集资源
- 资源管理系统
- 忠诚度系统

#### 4. 外交系统
- 贸易机制：交换资源、人口、技术
- 战争机制：军事实力计算、战斗、掠夺

#### 5. 游戏输出
- 详细的游戏状态显示
- 自动保存游戏状态
- 最终结算报告

### 游戏机制

#### 初始数值
- 资源：100
- 人口：50
- 军队：10
- 科技：1
- 忠诚度：100

#### 时代划分
- 原始时代：0-1技术
- 古代时代：2-4技术
- 中世纪：5-9技术
- 现代：10-14技术
- 未来：15+技术

#### 行动成本
- 发展科技：20资源/单位
- 建造军队：15资源/单位
- 增长人口：10资源/单位
- 收集资源：0资源/单位

### 配置说明

1. 在`config.json`中配置API密钥
2. 运行`python game.py`开始游戏
3. 游戏默认运行100回合
