# 版本 0.2.1 更新公告

发布日期：2026-03-14

## 主要更新

### 文化系统
- 新增文化属性和相关机制
- 文化高的文明影响文化低的文明
- 文化影响包括忠诚度影响和人口吸收
- 战争中可以劫掠对方文化

### Web界面优化
- 显示文明每一回合采取的行动
- 显示每个文明使用的AI模型
- 外交历史记录与图标
- 文明旗帜区分
- 资源图标改进
- 不同时代的文明图像（使用智谱清言生成）
- 时代事件显示

### 时代系统
- 时代精细化：原始时代、古典时代、中世纪时代、文艺复兴时代、工业时代、现代、信息时代和未来时代
- 时代AP调整：每隔两个新的时代才会给1个额外AP
- 时代跨越机制改进：加权考虑科技和文化，科技占比较大
- 时代事件系统：每个时代可能发生特定事件

### 游戏机制优化
- 战争对忠诚度的影响：进攻并获胜增加忠诚度，进攻并失败减少忠诚度
- 战争限制：战争发起者之后的1个Turn人口和资源的自然增长减半
- 减少战争防守方防守失败时的忠诚度降低值
- 游戏轮数增加：从10轮增加到50轮

### 视觉体验
- 添加了时代背景图片，增强视觉体验
- 图片使用智谱清言生成

## 技术更新
- 修复了Era Events显示问题
- 优化了时代事件触发机制
- 修复了时代跨越的计算逻辑
- 完善了静态文件服务

## 游戏截图

游戏现在具有更丰富的视觉效果，包括不同时代的背景图片，增强了游戏的沉浸感。

## 如何升级

直接运行最新版本的游戏即可，无需额外配置。

```bash
python web_gui.py
```

访问 http://localhost:8000 查看Web图形界面。
